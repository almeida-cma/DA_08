📚 Dicionário - POO e Construtores

Consulte sempre que tiver dúvida sobre o significado de uma palavra:

class → Classe (molde para objetos)
object → Objeto (instância da classe)
constructor → Construtor (inicializa objetos)
this → Este (refere-se ao próprio objeto)
new → Novo (cria um objeto)
attribute → Atributo (característica do objeto)
method → Método (ação do objeto)
encapsulation → Encapsulamento (protege dados)
inheritance → Herança (compartilha atributos)
polymorphism → Polimorfismo (mesma palavra, diferentes significados)
overload → Sobrecarga (vários métodos com mesmo nome)
default → Padrão (construtor sem parâmetros)
