🧩 Desafio 1: Classe Livro com Construtores

📌 Classe: DesafioLivro.java
📌 Atributos: titulo, autor, ano
📌 3 construtores: default, com título, com título e autor
📌 Método: exibirInfo()
📚 TRÍADE DIDÁTICA:
  • CONCEITO: Construtores sobrecarregados para diferentes formas de criar um livro.
  • OBJETIVO: Permitir criar livro com título, com título+autor, ou sem informações.
  • MÉTODO: Atributos recebem valores padrão quando não fornecidos.
📤 Saída esperada:
--- LIVRO 1 (default) ---
Título: Desconhecido
Autor: Desconhecido
Ano: 0

--- LIVRO 2 (com título) ---
Título: Java para Iniciantes
Autor: Desconhecido
Ano: 0

--- LIVRO 3 (com título e autor) ---
Título: POO em Java
Autor: Maria Silva
Ano: 0

📝 Código Sugerido:

package tema08;

public class DesafioLivro {
    // ATRIBUTOS
    String titulo;
    String autor;
    int ano;

    // CONSTRUTOR 1: default
    public DesafioLivro() {
        this.titulo = "Desconhecido";
        this.autor = "Desconhecido";
        this.ano = 0;
    }

    // CONSTRUTOR 2: com título
    public DesafioLivro(String titulo) {
        this.titulo = titulo;
        this.autor = "Desconhecido";
        this.ano = 0;
    }

    // CONSTRUTOR 3: com título e autor
    public DesafioLivro(String titulo, String autor) {
        this.titulo = titulo;
        this.autor = autor;
        this.ano = 0;
    }

    // MÉTODO para exibir informações
    public void exibirInfo() {
        System.out.println("Título: " + titulo);
        System.out.println("Autor: " + autor);
        System.out.println("Ano: " + ano);
    }

    // MAIN para teste
    public static void main(String[] args) {
        System.out.println("--- LIVRO 1 (default) ---");
        DesafioLivro livro1 = new DesafioLivro();
        livro1.exibirInfo();

        System.out.println("\n--- LIVRO 2 (com título) ---");
        DesafioLivro livro2 = new DesafioLivro("Java para Iniciantes");
        livro2.exibirInfo();

        System.out.println("\n--- LIVRO 3 (com título e autor) ---");
        DesafioLivro livro3 = new DesafioLivro("POO em Java", "Maria Silva");
        livro3.exibirInfo();
    }
}
🎯 O QUE FOI APRENDIDO:
  • ✅ Sobrecarga de construtores
  • ✅ Construtor default sem parâmetros
  • ✅ Construtores com parâmetros diferentes
  • ✅ Valores padrão para atributos não informados

🧩 Desafio 2: Classe Retangulo com Construtores

📌 Classe: DesafioRetangulo.java
📌 Atributos: largura, altura
📌 2 construtores: default (1x1) e parametrizado
📌 Métodos: calcularArea(), calcularPerimetro(), exibirInfo()
📚 TRÍADE DIDÁTICA:
  • CONCEITO: Construtor default e parametrizado para retângulos.
  • OBJETIVO: Criar retângulo com tamanhos padrão ou personalizados.
  • MÉTODO: Construtor default define 1x1, parametrizado recebe valores.
📤 Saída esperada:
--- RETANGULO 1 (default) ---
Largura: 1.0, Altura: 1.0
Área: 1.0, Perímetro: 4.0

--- RETANGULO 2 (parametrizado) ---
Largura: 5.0, Altura: 3.0
Área: 15.0, Perímetro: 16.0

📝 Código Sugerido:

package tema08;

public class DesafioRetangulo {
    // ATRIBUTOS
    double largura;
    double altura;

    // CONSTRUTOR 1: default (1x1)
    public DesafioRetangulo() {
        this.largura = 1.0;
        this.altura = 1.0;
    }

    // CONSTRUTOR 2: parametrizado
    public DesafioRetangulo(double largura, double altura) {
        this.largura = largura;
        this.altura = altura;
    }

    // MÉTODO para calcular área
    public double calcularArea() {
        return largura * altura;
    }

    // MÉTODO para calcular perímetro
    public double calcularPerimetro() {
        return 2 * (largura + altura);
    }

    // MÉTODO para exibir informações
    public void exibirInfo() {
        System.out.println("Largura: " + largura + ", Altura: " + altura);
        System.out.println("Área: " + calcularArea() + ", Perímetro: " + calcularPerimetro());
    }

    // MAIN para teste
    public static void main(String[] args) {
        System.out.println("--- RETANGULO 1 (default) ---");
        DesafioRetangulo r1 = new DesafioRetangulo();
        r1.exibirInfo();

        System.out.println("\n--- RETANGULO 2 (parametrizado) ---");
        DesafioRetangulo r2 = new DesafioRetangulo(5.0, 3.0);
        r2.exibirInfo();
    }
}
🎯 O QUE FOI APRENDIDO:
  • ✅ Construtor default com valores padrão
  • ✅ Construtor parametrizado para personalização
  • ✅ Métodos de cálculo (área e perímetro)
  • ✅ Encapsulamento de funcionalidades

🧩 Desafio 3: Classe Pessoa com this()

📌 Classe: DesafioPessoa.java
📌 Atributos: nome, idade, cidade
📌 3 construtores usando this() para reaproveitar código
📌 Método: exibirDados()
📚 TRÍADE DIDÁTICA:
  • CONCEITO: this() chama outro construtor da mesma classe.
  • OBJETIVO: Reaproveitar código e evitar repetição.
  • MÉTODO: Construtores menores chamam o construtor principal com valores padrão.
📤 Saída esperada:
--- PESSOA 1 (so nome) ---
Nome: Joao, Idade: 0, Cidade: Nao informada

--- PESSOA 2 (nome e idade) ---
Nome: Maria, Idade: 25, Cidade: Nao informada

--- PESSOA 3 (nome, idade e cidade) ---
Nome: Pedro, Idade: 30, Cidade: Sao Paulo

📝 Código Sugerido:

package tema08;

public class DesafioPessoa {
    // ATRIBUTOS
    String nome;
    int idade;
    String cidade;

    // CONSTRUTOR 1: com nome (usa this() para chamar o construtor 3)
    public DesafioPessoa(String nome) {
        this(nome, 0, "Nao informada");
    }

    // CONSTRUTOR 2: com nome e idade (usa this() para chamar o construtor 3)
    public DesafioPessoa(String nome, int idade) {
        this(nome, idade, "Nao informada");
    }

    // CONSTRUTOR 3: com nome, idade e cidade (construtor principal)
    public DesafioPessoa(String nome, int idade, String cidade) {
        this.nome = nome;
        this.idade = idade;
        this.cidade = cidade;
    }

    // MÉTODO para exibir dados
    public void exibirDados() {
        System.out.println("Nome: " + nome + ", Idade: " + idade + ", Cidade: " + cidade);
    }

    // MAIN para teste
    public static void main(String[] args) {
        System.out.println("--- PESSOA 1 (so nome) ---");
        DesafioPessoa p1 = new DesafioPessoa("Joao");
        p1.exibirDados();

        System.out.println("\n--- PESSOA 2 (nome e idade) ---");
        DesafioPessoa p2 = new DesafioPessoa("Maria", 25);
        p2.exibirDados();

        System.out.println("\n--- PESSOA 3 (nome, idade e cidade) ---");
        DesafioPessoa p3 = new DesafioPessoa("Pedro", 30, "Sao Paulo");
        p3.exibirDados();
    }
}
🎯 O QUE FOI APRENDIDO:
  • ✅ Uso de this() para reaproveitar construtores
  • ✅ Construtor principal que recebe todos os atributos
  • ✅ Construtores secundários com valores padrão
  • ✅ Reutilização de código e redução de duplicação
💡 DICA IMPORTANTE: this() deve ser a primeira instrução dentro do construtor!
