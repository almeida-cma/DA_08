📖 O que é um Construtor?

Um construtor é um método especial que é chamado automaticamente quando um novo objeto é criado. Ele serve para inicializar os atributos do objeto.

📚 TRÍADE DIDÁTICA:
  • CONCEITO: Método especial que inicializa objetos.
  • OBJETIVO: Garantir que o objeto comece com valores válidos.
  • MÉTODO: Mesmo nome da classe, sem retorno, chamado com new.
🎯 CARACTERÍSTICAS IMPORTANTES:
  • 📌 Tem o mesmo nome da classe
  • 📌 Não tem retorno (nem void)
  • 📌 Pode receber parâmetros (sobrecarga)
  • 📌 Toda classe tem pelo menos um construtor
  • 📌 Se não for definido, Java cria um construtor padrão (default)

🔧 Construtor Padrão (Default)

Se você não criar nenhum construtor, o Java cria automaticamente um construtor vazio (sem parâmetros).

// Construtor padrão criado automaticamente pelo Java
public class Carro {
    String modelo;
    // NENHUM construtor foi criado → Java cria um default vazio
}

// No main:
Carro carro = new Carro();  // Construtor padrão é chamado

📦 Construtor Parametrizado

Permite passar valores na hora de criar o objeto, já inicializando os atributos.

public class Carro {
    String modelo;
    String cor;

    // Construtor com parâmetros
    public Carro(String modelo, String cor) {
        this.modelo = modelo;
        this.cor = cor;
    }
}

// No main:
Carro carro = new Carro("Fusca", "Azul");  // Já com valores!

🔄 Sobrecarga de Construtores

Uma classe pode ter vários construtores com diferentes parâmetros.

public class Carro {
    String modelo;
    String cor;

    // Construtor 1: sem parâmetros (default)
    public Carro() {
        this.modelo = "Desconhecido";
        this.cor = "Branco";
    }

    // Construtor 2: com modelo
    public Carro(String modelo) {
        this.modelo = modelo;
        this.cor = "Branco";
    }

    // Construtor 3: com modelo e cor
    public Carro(String modelo, String cor) {
        this.modelo = modelo;
        this.cor = cor;
    }
}

💻 Exemplos Práticos

📌 Exemplo 1: Construtor Default e Parametrizado

🎯 OBJETIVO: Mostrar a diferença entre construtor padrão e construtor com parâmetros. 
package tema08;

public class Exemplo1Circulo {
    double raio;
    String cor;

    // Construtor padrão (default)
    public Exemplo1Circulo() {
        this.raio = 1.0;
        this.cor = "Branco";
    }

    // Construtor parametrizado
    public Exemplo1Circulo(double raio, String cor) {
        this.raio = raio;
        this.cor = cor;
    }

    public double calcularArea() {
        return Math.PI * raio * raio;
    }

    public void exibirInfo() {
        System.out.println("Raio: " + raio);
        System.out.println("Cor: " + cor);
        System.out.println("Área: " + calcularArea());
    }

    public static void main(String[] args) {
        System.out.println("--- CÍRCULO 1 (default) ---");
        Exemplo1Circulo c1 = new Exemplo1Circulo();
        c1.exibirInfo();

        System.out.println("\n--- CÍRCULO 2 (parametrizado) ---");
        Exemplo1Circulo c2 = new Exemplo1Circulo(5.0, "Vermelho");
        c2.exibirInfo();
    }
}
📤 Saída:
--- CÍRCULO 1 (default) ---
Raio: 1.0
Cor: Branco
Área: 3.141592653589793

--- CÍRCULO 2 (parametrizado) ---
Raio: 5.0
Cor: Vermelho
Área: 78.53981633974483

📌 Exemplo 2: Sobrecarga de Construtores

🎯 OBJETIVO: Demonstrar múltiplos construtores com diferentes parâmetros. 
package tema08;

public class Exemplo2Aluno {
    String nome;
    int idade;
    String curso;

    // Construtor 1: sem parâmetros
    public Exemplo2Aluno() {
        this.nome = "Não informado";
        this.idade = 0;
        this.curso = "Não informado";
    }

    // Construtor 2: com nome
    public Exemplo2Aluno(String nome) {
        this.nome = nome;
        this.idade = 0;
        this.curso = "Não informado";
    }

    // Construtor 3: com nome e idade
    public Exemplo2Aluno(String nome, int idade) {
        this.nome = nome;
        this.idade = idade;
        this.curso = "Não informado";
    }

    // Construtor 4: com nome, idade e curso
    public Exemplo2Aluno(String nome, int idade, String curso) {
        this.nome = nome;
        this.idade = idade;
        this.curso = curso;
    }

    public void exibirInfo() {
        System.out.println("Nome: " + nome);
        System.out.println("Idade: " + idade);
        System.out.println("Curso: " + curso);
    }

    public static void main(String[] args) {
        System.out.println("--- ALUNO 1 (default) ---");
        Exemplo2Aluno a1 = new Exemplo2Aluno();
        a1.exibirInfo();

        System.out.println("\n--- ALUNO 2 (só nome) ---");
        Exemplo2Aluno a2 = new Exemplo2Aluno("João");
        a2.exibirInfo();

        System.out.println("\n--- ALUNO 3 (nome e idade) ---");
        Exemplo2Aluno a3 = new Exemplo2Aluno("Maria", 20);
        a3.exibirInfo();

        System.out.println("\n--- ALUNO 4 (nome, idade e curso) ---");
        Exemplo2Aluno a4 = new Exemplo2Aluno("Pedro", 22, "Java");
        a4.exibirInfo();
    }
}

--- ALUNO 3 (nome e idade) ---
Nome: Maria
Idade: 20
Curso: Não informado

📌 Exemplo 3: this() para chamar outro construtor

🎯 OBJETIVO: Mostrar como um construtor pode chamar outro usando this(). 
package tema08;

public class Exemplo3Produto {
    String nome;
    double preco;
    int quantidade;

    // Construtor com todos os parâmetros
    public Exemplo3Produto(String nome, double preco, int quantidade) {
        this.nome = nome;
        this.preco = preco;
        this.quantidade = quantidade;
    }

    // Construtor com nome e preco (quantidade padrão = 0)
    public Exemplo3Produto(String nome, double preco) {
        this(nome, preco, 0);  // Chama o construtor acima!
    }

    // Construtor só com nome (preco padrão = 0, quantidade = 0)
    public Exemplo3Produto(String nome) {
        this(nome, 0.0, 0);  // Chama o construtor acima!
    }

    public void exibirInfo() {
        System.out.println("Nome: " + nome);
        System.out.println("Preço: R$ " + preco);
        System.out.println("Quantidade: " + quantidade);
    }

    public static void main(String[] args) {
        System.out.println("--- PRODUTO 1 (nome, preco, quantidade) ---");
        Exemplo3Produto p1 = new Exemplo3Produto("Notebook", 2500.0, 10);
        p1.exibirInfo();

        System.out.println("\n--- PRODUTO 2 (nome, preco) ---");
        Exemplo3Produto p2 = new Exemplo3Produto("Mouse", 50.0);
        p2.exibirInfo();

        System.out.println("\n--- PRODUTO 3 (só nome) ---");
        Exemplo3Produto p3 = new Exemplo3Produto("Teclado");
        p3.exibirInfo();
    }
}
📤 Saída:
--- PRODUTO 1 (nome, preco, quantidade) ---
Nome: Notebook
Preço: R$ 2500.0
Quantidade: 10

--- PRODUTO 2 (nome, preco) ---
Nome: Mouse
Preço: R$ 50.0
Quantidade: 0

--- PRODUTO 3 (só nome) ---
Nome: Teclado
Preço: R$ 0.0
Quantidade: 0
